## 🌟 About NayePankh

NayePankh Foundation is a youth-driven organization working towards creating meaningful change in underserved communities across India. From organizing awareness campaigns and community drives to mobilizing volunteers and managing donations, every initiative is powered by people who believe in making a difference.

As the foundation grew, so did the need for a simple way to organize information. This project was created to help streamline day-to-day operations and maintain records efficiently.

---

## 💙 Why This Project?

Managing a social impact organization involves much more than conducting campaigns. Volunteer details, donor contributions, and campaign records need to be maintained carefully to ensure transparency and smooth operations.

The **NayePankh Foundation Management System** is a lightweight Python-based application designed to simplify these tasks and keep important information organized in one place.

---

## ✨ Features

### 🙋 Volunteer Management

Maintain a database of volunteers, including their contact information and roles within the organization.

### 🎁 Donation Tracking

Record and manage financial as well as in-kind donations such as food, clothing, educational materials, and other essentials.

### 📢 Campaign Management

Store details of campaigns conducted by the foundation, including location, purpose, and community impact.

### 🗄️ Centralized Data Storage

All information is stored locally, making the system simple, reliable, and easy to use without requiring internet connectivity.

## 🚀 Getting Started

### Prerequisites

Ensure that you have **Python 3.x** installed on your system.

Verify installation:

```bash
python --version
```

### Run the Application

Clone the repository and navigate to the project directory:

```bash
git clone <repository-url>
cd nayepankh
```

Start the application:

```bash
python main.py
```

---

## 📋 Using the System

After launching the application, an interactive menu will appear.

You can:

* View and manage volunteers
* Track donations received
* Record and review campaigns
* Access foundation data through a simple menu-driven interface

Just follow the on-screen instructions to navigate through the system.
